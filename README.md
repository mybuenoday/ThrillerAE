2. 팀명 Thriller Æ
스릴러와 같은 감동과 긴장감을 선사하는 프로젝트를 만드는 팀.
Æ는 인공지능과 사랑을 상징하는 기호
Æ는 라틴 문자로 'ash’라고 읽습니다. 그리스 문자로는 'æta’라고 읽습니다.
일론 머스크와 그라임스의 아들의 이름에서는 'ash’로 읽는다고 합니다.
